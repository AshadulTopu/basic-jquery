fadeIn() fadeOut() fadeToggle() fadeTo()

fadeIn() fadeOut() fadeToggle() fadeTo() is jQuery build in method.
fadeIn() use for invisible to visible thing with fade effect
fadeOut() use for visible to invisible thing with fade effect
fadeToggle() use for visible and invisible thing with fade effect
fadeTo() fadeTo like fade , but using callback function we can do many thing
we can use callback function with all method
